### 3. 多窗口确认优化

**文件：** `polymarket_sdk/services/smart_money_service_v2.py`

#### 3.1 分层确认机制

**配置：**
```bash
# Level 1（立即）
LARGE_TRADE_IMMEDIATE_THRESHOLD=5000      # 单笔 >= $5000
LARGE_FLOW_IMMEDIATE_THRESHOLD=10000      # 净流入 >= $10000

# Level 2（快速）
MEDIUM_FLOW_FAST_THRESHOLD=3000           # 净流入 >= $3000
MEDIUM_TRADER_FAST_THRESHOLD=5            # 交易员 >= 5

# Level 3（标准）
REQUIRED_CONSECUTIVE_WINDOWS=3            # 需要3次窗口确认
```

**逻辑：**
- **Level 1（立即）：** 单笔 >= $5000 或 净流入 >= $10000 → 立即生成信号
- **Level 2（快速）：** 净流入 >= $3000 且 交易员 >= 5 → 需要2次窗口确认
- **Level 3（标准）：** 默认 → 需要3次窗口确认

#### 3.2 空窗口检查

**问题：**
- 窗口1有交易 → Window 1/3 ✅
- 窗口2没有交易 → 不触发检查 ❌
- 窗口3没有交易 → 不触发检查 ❌
- 结果：永远无法达到 Window 3/3

**修复：**
```python
# 之前
if window_elapsed >= 60 and len(buffer['trades']) > 0:
    # 只有窗口内有交易才检查

# 之后
if window_elapsed >= 60:
    # 无论有没有交易都检查
    if len(buffer['trades']) > 0:
        # 有交易 → 正常处理
    else:
        # 空窗口 → 信号消失 → 重置确认计数
```

**效果：**
- ✅ 系统会定期检查所有窗口（无论有没有交易）
- ✅ 空窗口会重置确认计数
- ✅ 修复了 Window 2/3 和 Window 3/3 无法触发的问题

### 4. Copytrading V2 优化

**文件：** `skills_v2/copytrading_v2.py`

#### 4.1 查询所有交易员

**修改：**
```python
# 之前
for i, trader in enumerate(qualified_traders[:10], 1):  # 只查询前10个

# 之后
for i, trader in enumerate(qualified_traders, 1):  # 查询所有交易员
```

#### 4.2 缓存机制

**新增：**
```python
# 缓存交易员列表（1小时更新一次）
self._cached_traders = []
self._cache_time = 0
self._cache_ttl = 3600  # 1小时

async def _get_qualified_traders(self, max_traders=200):
    # 如果缓存未过期，直接返回
    if self._cached_traders and (now - self._cache_time) < self._cache_ttl:
        return self._cached_traders

    # 否则重新获取
    leaderboard = await self.data_api.get_all_leaderboard(limit=max_traders)
    qualified_traders = [t for t in leaderboard if t.get('pnl', 0) >= 100]

    # 更新缓存
    self._cached_traders = qualified_traders
    self._cache_time = now

    return qualified_traders
```

#### 4.3 降低门槛

**修改：**
```python
# 之前
min_trader_pnl = 1000  # PnL >= $1000

# 之后
min_trader_pnl = 100   # PnL >= $100
```

**效果：**
- ✅ 查询所有交易员（不再限制为10个）
- ✅ 缓存机制减少 API 调用
- ✅ 降低门槛，更多交易员符合条件
