# Polymarket 交易策略最佳实践

## 核心参数配置（基于搜索结果）

### 1. 过滤阈值
| 参数                 | 推荐值         | 说明     |
| ------------------ | ----------- | ------ |
| MIN_NET_FLOW       | $2000+      | 净流入门槛  |
| MIN_LIQUIDITY      | $10000      | 最小流动性  |
| MIN_TIME_TO_EXPIRY | 6-12h       | 距离到期时间 |
| PRICE_RANGE        | $0.50-$0.99 | 价格区间   |

### 2. Smart Money 指标
- **PNL** - 盈亏
- **ROI** - 回报率  
- **交易频率** - 区分套利/做市 vs 方向交易

### 3. 策略分类
- **Sports**: 严格模式 (2倍门槛)
- **Politics/Crypto**: 标准模式
- **低价区 (<$0.50)**: 跳过

### 4. 关键发现
1. 低价流动性差，卖出困难
2. 高价区($0.90-$0.99)快到期时波动大
3. 政治类胜率高于体育类
4. 需要结合自己的研究，不盲目跟单

### 5. 优化方向
- 启用 category 过滤
- 价格区间严格检查
- Sports 严格模式
- 动态净流入阈值
