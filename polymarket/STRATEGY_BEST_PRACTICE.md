## 📊 市场结构理解

### Polymarket 二元期权机制

每个预测问题有**两个 Token**：

| Token | 含义 | 结算价值 |
|-------|------|---------|
| **YES** | "这件事会发生" | 结果为 YES = $1.00，否则 $0.00 |
| **NO** | "这件事不会发生" | 结果为 NO = $1.00，否则 $0.00 |

**关键公式**：
```
YES 价格 + NO 价格 ≈ $1.00
```

## 🎯 七大核心策略

### 1. NO Farming（做空过度炒作）

**原理**：大多数预测市场最终结果是 NO

**操作**：
- 筛选 YES 价格 $0.80-0.97 的市场
- 买入 NO token（做空 YES）
- 等待回归或结算

**胜率因素**：
- YES 越高越好（群众越乐观，越可能错）
- 时间越短越好（减少不确定性）
- 政治/体育类更好（容易炒作）

**风险**：黑天鹅事件（罕见奇迹发生）

---

### 2. High-Probability Compounding（高概率复利）

**原理**：买入高概率合约，小额频繁交易

**操作**：
- 买入 YES @ $0.90-0.99
- 胜率 90%+，利润 1-10%
- 高频重复，复利增长

**适合**：
- 短期市场（< 24 小时）
- 体育比赛、加密价格

---

### 3. Spread Farming（价差套利）

**原理**：利用买卖价差异

**操作**：
- 监控 YES + NO < $1.00 的机会
- 同时买入双方
- 结算时保证盈利

**注意**：
- 考虑 3.15% 手续费
- 需要快速执行

---

### 4. Long-Shot Floor Buying（低价彩票）

**原理**：买入极端低价合约，博取不对称收益

**操作**：
- 买入 YES @ $0.01-0.05
- 亏损有限，收益无限
- 分散到 100+ 市场

**风险**：大多数归零，需要资金分散

---

### 5. Structural Spread Lock（结构价差锁定）

**原理**：利用订单簿失衡

**操作**：
- 发现恐慌性抛售
- 同时买入 YES + NO
- 锁定价差

---

### 6. Liquidity Absorption Flip（流动性吸收翻转）

**原理**：大资金操纵市场

**操作**：
- 吸收流动性建立仓位
- 结算前推动参考价格
- 翻转结果获利

**风险**：高资金要求，道德灰色

---

### 7. Copy Trading（跟单交易）

**原理**：跟随高胜率钱包

**操作**：
- 跟踪 Top 50 高胜率地址
- 检测共识（多人同方向）
- 跟随买入

**关键参数**：
- `min_trader_count`：共识人数（3-5人）
- `min_net_flow`：净流入阈值（$3000-$5000）
- `min_trader_quality`：交易员质量分（70+）

## 📉 死亡区间

### 价格区间胜率分析

| 价格区间 | 胜率 | 建议 |
|----------|------|------|
| $0.00-0.30 | 高风险 | 适合 Long-Shot |
| **$0.30-0.50** | 中等 | ⚠️ 谨慎 |
| **$0.50-0.70** | **最低** | **❌ 避免** |
| **$0.70-0.80** | **低** | **❌ 死亡区间** |
| $0.80-0.97 | 高 | ✅ NO Farming |
| $0.97-1.00 | 很高 | ✅ 高概率复利 |

**关键发现**：
- **$0.70-0.80 是死亡区间**（胜率仅 25%）
- **$0.80+ 是 NO Farming 黄金区**

## 🔧 风控原则

### 仓位管理

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 单笔金额 | $5-10 | 小额分散 |
| 最大持仓数 | 20 | 避免过度分散 |
| 单市场最大 | $50 | 限制敞口 |
| 止损 | -15% | 严格执行 |

### 信号过滤

| 过滤器 | 阈值 | 原因 |
|--------|------|------|
| 置信度 | ≥ 60% | 过滤噪音 |
| 价差 | ≤ 3% | 流动性检查 |
| 流动性 | ≥ $10,000 | 价格准确 |
| 到期时间 | 1-720小时 | 不太长不太短 |

---

## 🚫 黑名单市场类型

**避免交易**：
- BTC 价格类（"BTC 会涨到 $X"）— 难预测
- Spread/Over-Under — 庄家优势大
- "Up or Down" 类 — 纯赌博
- 极长期市场（> 30 天）— 不确定性高

## 🎯 优化建议

### 1. 禁用亏损策略
- Smart Money 大资金监控持续亏损
- 设高门槛或直接禁用

### 2. 启用 NO Farming
- 目标：YES $0.80-0.97
- 操作：买入 NO token
- 预期：稳定正收益

### 3. 减少持仓数量
- 当前 88 个过多
- 建议控制在 20 个以内

### 4. 专注高胜率区间
- 优先 $0.80+ 市场
- 避开 $0.60-0.80 死亡区间
